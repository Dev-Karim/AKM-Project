About This Project

This project was built as part of CMPS 242 — Computer Networks at the American University of Beirut (AUB).

How It Works

The server is implemented in a single Java file (WebServer.java) containing two classes:

  • WebServer — binds a ServerSocket to port 6789 and accepts client connections in a loop.
  • HttpRequest — parses one HTTP GET request, locates the file, and sends back a complete HTTP/1.0 response.

Multithreading

Each accepted connection is wrapped in an HttpRequest object and started in a new Thread, so multiple clients can be served simultaneously without one blocking another.
